# React Task Manager

## Features

- Drag and drop tasks between *To Do*, *In Progress*, and *Done* columns.
- Vite for instant server starts.
- Tasks save automatically to 'localStorage'.
- React Context API (No external heavy libraries).
- strict TypeScript configuration.

## Tech Stack

- Build Tool: Vite
- Framework: React 19
- Language: TypeScript
- Styling: Tailwind CSS

## Setup Instructions

### 1. Prerequisites
- Node.js (v18+)

### 2. Clone & Install

git clone https://github.com/gp0811/task-manager.git
cd task-manager
npm install

3. Run Locally
npm run dev
Open http://localhost:3000 to view the app.

4. Build for Production
Bash
npm run build
npm run preview
